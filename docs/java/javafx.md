# JavaFX

- Site officiel: [openjfx.io](https://openjfx.io/)
- Créer une projet avec JBang : `jbang init --template javafx@yostane jfx.java`

## Démo

```java
--8<--
javafx/HellojfxApp.java
--8<--
```
